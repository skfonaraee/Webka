# BookClub

**Тема:** Онлайн-библиотека / книжный клуб

## Описание проекта
BookClub — это многостраничный web-сайт онлайн-библиотеки и книжного клуба.
Проект демонстрирует использование семантического HTML, архитектуры CSS
и адаптивной вёрстки без применения JavaScript и фреймворков.

## Страницы сайта
- Главная
- О проекте
- Каталог
- Контакты

## Используемые технологии
- HTML5
- CSS3 (Flexbox, Grid)
